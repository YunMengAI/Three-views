# YM-角色卡

这是一个 ComfyUI 人物角色卡提示词插件。当前阶段只输出提示词，用来测试角色卡相关提示词质量。

节点在 ComfyUI 里显示为：`角色卡 提示词生成器`

分类显示为：`YM-角色卡/提示词`

## 安装方式

把这个仓库下载或克隆到 ComfyUI 的 `custom_nodes` 目录里即可：

```text
ComfyUI/custom_nodes/Three-views
```

然后重启 ComfyUI，在节点搜索里搜索：

```text
角色卡
YM-角色卡
```

## 功能

支持生成 4 类提示词：

- 人物三视图
- 人物面部三视图
- 人物面部增强三视图
- 人物角色卡

支持输入：

- 面部参考图，可选
- 服装参考图，可选
- 角色简介/生成要求
- RH LLM 模型选择
- 视觉风格预设和自定义视觉风格

## 说明

本插件不是生图节点，不调用 gpt-image2.0。它只负责把参考图和文字要求整理成适合后续生图节点使用的提示词。

系统提示词单独放在：

```text
prompt_templates/
```

需要调整提示词规则时，直接修改对应 `.txt` 文件即可。
